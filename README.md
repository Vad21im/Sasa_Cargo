# Sasa_Cargo 
Проект на стадии прототипа.
Проект создан для упращения работы  службе логистики онлайн площадки  Sasa.ru
На  данный момент заморожен со стороны заказчика.
Есть в проекти некоторые недостатки такие как прямая мутация стейта, это было сделано для ускорения сдачи прототипа и в случае дальнешего развития проекта будет фиксится.
